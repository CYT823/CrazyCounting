# CrazyCounting
JAVA game - guessing the safe box

Language: JAVA GUI

Tools: eclipse

Owner: CYT

------------------------------------------------------

Classes:

GUI: The execute class.

Util: The class contains functions.

CrazyCountingFrame, RuleFrame: JAVA GUI

------------------------------------------------------

Rules:

透過邊上的提示，找出安全的格子。

棋盤邊上是提示，告訴玩家從該方向開始連續藍色格子的個數。

例如：12，表示這一列(行)共有3個安全的格子，先是1個，然後是2個相連。
